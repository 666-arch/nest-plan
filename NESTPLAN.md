## Nest Cli
```bash
# 全局安装脚手架
$ npm install -g @nestjs/cli
```

```bash
# create project
$ nest new 项目名
```

```bash
# version update: last version
$ npm update -g @nestjs/cli
```

```bash
# create all the module - Http Rest Api
$ nest generate resource 模块名称
```

```bash
# build project
$ nest build
```
